TypeScript是由微软开发的一种开源编程语言。它是JavaScript的超集，增加了可选的静态类型和基于类的面向对象编程。TypeScript可以编译成纯JavaScript，在任何浏览器、任何主机和任何操作系统上运行。

对比JavaScript，TypeScript增加的特性有：

- 类型批注和编译是类型检查
- 类型推断
- 类型擦除
- 接口
- 枚举
- Mixin
- 泛型编程
- 命名空间
- 元组
- Await
