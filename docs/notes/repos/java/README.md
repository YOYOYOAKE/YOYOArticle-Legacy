Java 作为世界上最流行的编程语言之一，在 Web 开发中有着举足轻重的地位。

夸张一点说，Java 就是 Web 后端，Web 后端就是 Java。

除了网络开发，Java 还几乎成为了面向对象编程的代名词。

尽管在 Web 后端已经有了 C#、Rust、Golang 等富有竞争力的语言，但是 Java 仍然值得一学。
